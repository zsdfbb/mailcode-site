Contrôlez OpenCode
Où Que Vous Soyez, Par Email

MailCode connecte votre boîte mail à OpenCode — l'agent de codage IA open source. Envoyez des tâches, recevez des résultats et injectez des commandes via n'importe quel compte email. Zéro dépendance tierce. Une seule commande à installer.

Comment ça marche

MailCode crée un pont simple entre votre boîte mail et OpenCode tournant dans tmux.

✉️

Vous envoyez un email

Une tâche de codage depuis n'importe quel client mail — téléphone, tablette ou ordinateur.

→
📡

IMAP Listener

MailCode surveille votre boîte en temps réel via IMAP IDLE, détectant les nouveaux messages instantanément.

→
⚡

Injection tmux

Les commandes sont injectées dans votre session OpenCode via tmux send-keys.

→
📬

Notification

Une fois la tâche terminée, vous recevez un résumé par email — répondez pour continuer.

Deux façons de l'utiliser

Qu'OpenCode soit déjà en cours d'exécution ou non, MailCode a la solution.

Mode Réponse

Quand OpenCode est déjà en cours

OpenCode termine une tâche dans tmux. Le plugin pont détecte l'état inactif et vous envoie une notification par email avec un token unique de 8 caractères. Répondez simplement à cet email avec votre prochaine instruction.

Votre réponse : #A3B7KM9Q Continuer l'optimisation de parse_config
Mode Démarrage à Froid

Quand OpenCode n'est pas lancé

Envoyez un nouvel email avec votre session key et la description de la tâche. MailCode crée automatiquement une session tmux, lance OpenCode dans le répertoire du projet, et injecte votre commande.

Votre email (nouveau message): key: a1b2c3d4e5f67890 project: /path/to/your-project Refactoriser parse_config dans helper.py

Plusieurs couches de protection garantissent que vous seul contrôlez vos sessions OpenCode.

Liste Noire de Commandes

Liste Blanche d'Expéditeurs

Installer en une commande

Nécessite Python 3.9+, tmux et OpenCode CLI. Choisissez votre plateforme.

brew install python3 tmux
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

Après installation, éditez ~/.config/mailcode/config.json puis exécutez mailcode serve --idle.

apt update && apt install python3 tmux -y
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

Après installation, éditez ~/.config/mailcode/config.json puis exécutez mailcode serve --idle.

dnf install python3 tmux -y
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

Après installation, éditez ~/.config/mailcode/config.json puis exécutez mailcode serve --idle.

pacman -S python tmux --noconfirm

Scénarios d'utilisation

Les développeurs utilisent MailCode pour rester connectés à leur code où qu'ils soient.

🔍

Revue de Code à Distance

Le pipeline CI échoue quand vous êtes absent. MailCode vous notifie par email, vous répondez avec une correction, OpenCode l'exécute — depuis votre téléphone.

🔄

Automatisation Planifiée

Configurez des déclencheurs cron-like pour les sauvegardes nocturnes, mises à jour de dépendances ou refactoring automatisé.

🚨

Correctifs d'Urgence

Incident de production dans les transports ? Envoyez un email avec la commande de correctif. MailCode démarre OpenCode sur votre serveur, applique le correctif et confirme le résultat.

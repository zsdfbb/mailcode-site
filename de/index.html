Zwei Nutzungsmodi

Egal, ob Ihr KI-Agent bereits läuft oder nicht — MailCode ist bereit.

Antwortmodus

Agent läuft bereits

Der Agent sendet nach Abschluss eine E-Mail mit einem eindeutigen Token. Antworten Sie mit Ihrem nächsten Befehl.

Ihre Antwort-E-Mail: #A3B7KM9Q Optimierung der parse_config-Funktion fortsetzen
Kaltstart

Kein Agent aktiv

Senden Sie eine neue E-Mail mit Ihrem Sitzungsschlüssel. MailCode erstellt eine tmux-Sitzung und startet den Agenten.

Neue E-Mail: key: a1b2c3d4e5f67890 project: /path/to/your-project Refaktorieren Sie parse_config in helper.py

Installation mit einem Befehl

Erfordert Python 3.9+, tmux und ein KI-Agent-CLI.

pip install mailcode
mailcode serve --idle

Nach der Installation: mailcode config init dann mailcode serve --idle.

apt update && apt install python3 tmux -y
pip install mailcode
mailcode serve --idle

Nach der Installation: mailcode config init dann mailcode serve --idle.

dnf install python3 tmux -y
pip install mailcode
mailcode serve --idle

Nach der Installation: mailcode config init dann mailcode serve --idle.

pacman -S python tmux --noconfirm
pip install mailcode
mailcode serve --idle

# mailcode-site

MailCode 产品推广页面 — 控制 OpenCode AI 编程助手，通过邮件远程操控。

## 语言支持

| 语言 | 路径 | 状态 |
|------|------|------|
| English | `/en/` | ✅ |
| 中文 | `/zh/` | ✅ |
| 日本語 | `/ja/` | ✅ |
| 한국어 | `/ko/` | ✅ |
| Français | `/fr/` | ✅ |
| Deutsch | `/de/` | ✅ |

`/index.html` 自动检测浏览器语言并跳转到对应版本。

## 开发

纯 HTML/CSS/JS，零构建步骤。直接在浏览器中打开即可预览。

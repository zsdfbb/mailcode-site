OpenCode AI 编程助手

MailCode 将你的邮箱与 OpenCode 打通——开源 AI 编程助手。收发任务、注入命令、接收结果，全部通过邮件完成。零第三方依赖，一条命令完成安装。

工作原理

MailCode 在邮箱收件箱与 tmux 中的 OpenCode 之间搭建了一座简单的桥梁。

✉️

发送邮件

从手机、平板或笔记本发送编程任务或指令到你的邮箱。

→
📡

IMAP 实时监听

MailCode 通过 IMAP IDLE 长连接实时监听收件箱，新邮件秒级响应。

→
⚡

tmux 命令注入

命令通过 tmux send-keys 注入到正在运行的 OpenCode 会话中。

→
📬

邮件通知

任务完成后自动发送邮件通知，回复邮件即可继续下一轮操作。

两种使用模式

无论 OpenCode 是否已在运行，MailCode 都有应对方案。

回复模式

OpenCode 已在运行

OpenCode 完成任务后，桥接插件检测到空闲状态，自动发送邮件通知（含 8 位 Token）。直接回复该邮件即可输入下一条指令——Token 将命令路由到正确的会话，MailCode 通过 tmux send-keys 注入执行。

你的邮件回复： #A3B7KM9Q 继续优化 parse_config 函数
冷启动模式

OpenCode 尚未运行

发送一封新邮件，附上 session_key 和任务描述。MailCode 自动创建 tmux 会话，在目标项目中启动 OpenCode，并注入你的命令。收到确认邮件后即可切换到回复模式继续。

你的邮件（新邮件）： key: a1b2c3d4e5f67890 project: /path/to/your-project 重构 helper.py 中的 parse_config 函数

一条命令完成安装

需要 Python 3.9+、tmux 和 OpenCode CLI。选择你的平台。

brew install python3 tmux
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

安装完成后编辑 ~/.config/mailcode/config.json 填入邮箱配置，然后运行 mailcode serve --idle。

apt update && apt install python3 tmux -y
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

メールで OpenCode を
遠隔操作

MailCode はあなたのメールボックスと OpenCode（オープンソース AI コーディングエージェント）を橋渡しします。タスクの送信、結果の受信、コマンドの注入をすべてメール経由で。サードパーティ依存ゼロ、1コマンドでインストール。

仕組み

MailCode はメールボックスと tmux 上の OpenCode をシンプルに橋渡しします。

✉️

メールを送信

スマホやタブレットからプログラミングタスクをメールで送信。

📡

IMAP リスナー

IMAP IDLE 長期接続で受信トレイをリアルタイム監視。

→
⚡

tmux 注入

コマンドを tmux send-keys で OpenCode セッションに注入。

→
📬

通知

タスク完了時にメール通知。返信で次の操作に進めます。

2つの使用モード

OpenCode の実行状態に応じて最適な方法を提供します。

返信モード

OpenCode 実行中

OpenCode がタスクを完了すると、ブリッジプラグインがアイドル状態を検出し、8文字のトークン付きメール通知を送信。そのメールに返信するだけで次の指示を送れます。トークンが正しいセッションにルーティングします。

あなたの返信： #A3B7KM9Q parse_config 関数の最適化を続けて
コールドスタート

OpenCode 未実行

session_key とタスクを記載した新規メールを送信。MailCode が自動的に tmux セッションを作成し、対象プロジェクトで OpenCode を起動、コマンドを注入。確認メール受信後は返信モードに移行。

新規メール： key: a1b2c3d4e5f67890 project: /path/to/your-project helper.py の parse_config をリファクタリング

MailCode の特徴

どこにいてもコードをコントロールしたい開発者のために。

📡

リアルタイム IMAP IDLE

ポーリング遅延なし。IMAP IDLE 長期接続でメールを即座に検出。

🔐

DKIM + SPF 認証

DNS レベルのメール認証で送信者を検証。厳格モードでは未検証を拒否。

📢

マルチチャンネル通知

メール、macOS 通知、Slack、Discord、Feishu、DingTalk、WeCom、カスタム Webhook。

⏱

スケジュールタスク

Cron ライクなメールスケジューリングで定期自動化を実現。

💻

CLI + TUI 二重インターフェース

Textual ベースのターミナル UI と強力な CLI の両方を提供。

📦

サードパーティ依存ゼロ

純粋な Python 標準ライブラリのみ。pip install 不要。Python 3.9+ と tmux だけで動作。

セキュリティ最優先

複数層の保護により、あなただけが OpenCode セッションを制御できます。

🚫

コマンドブラックリスト

rm -rf、sudo rm、パイプ実行などをデフォルトでブロック。

✅

送信者ホワイトリスト

許可したメールアドレスのみコマンドを送信可能。それ以外は無視。

🔑

Session Key 認証

コールドスタートには 128 ビットのランダム session_key が必要。

⏳

セッション期限と制限

セッションは 24 時間で自動期限切れ。コマンド実行回数制限あり。

1コマンドでインストール

必要なもの：Python 3.9+、tmux、OpenCode CLI。プラットフォームを選択。

brew install python3 tmux
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

インストール後、~/.config/mailcode/config.json を編集し、mailcode serve --idle を実行。

apt update && apt install python3 tmux -y
git clone https://github.com/zsdfbb/mailcode.git
cd mailcode
bash install.sh

インストール後、~/.config/mailcode/config.json を編集し、mailcode serve --idle を実行。

活用シナリオ

開発者は MailCode を使ってどこにいてもコードベースに接続できます。

🔍

リモートコードレビュー

離席中に CI パイプラインが失敗。MailCode がメール通知、返信で修正コマンドを送信、OpenCode が処理——すべてスマホから。

🔄

定期自動化

Cron ライクなメールトリガーで夜間バックアップや依存関係更新を自動実行。

🚨

緊急ホットフィックス

通勤中に本番障害？メールでホットフィックスコマンドを送信。MailCode が自宅サーバーで OpenCode を起動、パッチ適用し結果を通知。

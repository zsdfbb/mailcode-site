仕組み

MailCode はメールと tmux 上の AI エージェントをシンプルに橋渡しします。

✉️

メール送信

スマホやタブレットからコーディングタスクをメールで送信。

→
📡

IMAP 受信

IMAP IDLE 長接続でリアルタイム受信、新メールを即時検出。

→
⚡

tmux 注入

tmux send-keys 経由で AI エージェントセッションにコマンド注入。

→
📬

通知受信

タスク完了時にメールまたは Webhook で通知。返信で続行。

2 つの利用モード

AI エージェントの状態に関わらず、MailCode が対応します。

返信モード

エージェント稼働中

エージェント完了後にユニーク Token 付きメール通知。返信で次の指示を送信。Token が正しいセッションにルーティング。

返信メール: #A3B7KM9Q parse_config 関数の最適化を続けて
コールドスタート

エージェント未起動

session_key 付き新規メールでエージェントを遠隔起動。tmux セッション作成、AI 起動、コマンド注入まで自動化。

新規メール: key: a1b2c3d4e5f67890 project: /path/to/your-project Refactor helper.py's parse_config

ワンコマンドでインストール

Python 3.9+、tmux、AI Agent CLI が必要です。

pip install mailcode
mailcode serve --idle

インストール後 mailcode config init でメール設定を行い、mailcode serve --idle でリレーを開始。

apt update && apt install python3 tmux -y
pip install mailcode
mailcode serve --idle

インストール後 mailcode config init でメール設定を行い、mailcode serve --idle でリレーを開始。

dnf install python3 tmux -y
pip install mailcode
mailcode serve --idle

インストール後 mailcode config init でメール設定を行い、mailcode serve --idle でリレーを開始。

pacman -S python tmux --noconfirm
pip install mailcode
mailcode serve --idle
